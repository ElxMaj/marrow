# Standup: sessions

We talked through session lifetime for the desk app. Because a terminal is shared and people walk away mid-shift, a session that lives forever is a real risk: the next person inherits the last person's access.

We agreed sessions expire after 8 hours, and a desk terminal also locks after 15 minutes idle so a walk-away does not leave an open session. Re-entry is a fresh magic link, which keeps the audit trail honest.

Mobile is different and we did not decide it here; treat mobile session length as a separate open thread.
